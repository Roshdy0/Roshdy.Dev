## 🧪 Mini Case Study: Engineering Methodology

### The Objective
To build a feature-rich gaming hardware store that maintains high performance while delivering complex, interactive user interfaces.

### The Implementation
* **Clean Code & Modularity:** Achieved through **Next.js dynamic imports** to optimize bundle sizes and ensure modular component loading.
* **Performance:** Leveraged code splitting and custom React Hooks to ensure low Total Blocking Time (TBT) and high Lighthouse scores.
* **Modern UI:** Integrated **Framer Motion** for smooth, trending animations that enhance the gaming aesthetic and user engagement.
